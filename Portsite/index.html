Introduction

This project was done to learn seeking behavior.
  • The pink dot indicates Target for AI which can be assigned using mouse click on any location.
  • The green dot is the AI with seeking behavior. It will seek the target and orbit around it.
The AI contains:
  • Acceleration
  • Velocity
  • Mass
  • Speed

Seeking and Arriving Behavior in Class

This project was done to learn seeking behavior.
  • The pink dot indicates Target for AI which can be assigned using mouse click on any location.
  • The green dot is the AI with seeking behavior. It will seek the target with in the line of sight. It orbit around traget a bit as it does not know when to stop.
The AI contains:
  • Acceleration
  • Velocity
  • Mass
  • Speed

Flee behavior

This project done to learn flee behavior.
  • The pink dot indicates Target for AI which can be assigned using mouse click on any location.
  • The white dot is the AI with flee behavior. It will flee the traget with in the line of sight. It will stop fleeing when the traget is out of line of sight.
The AI contains:
  • Acceleration
  • Velocity
  • Mass
  • Speed

Projectile

    The user clicks on any location on the screen to create an angle with the floor. When clicked, the green drop will launch according to the angle created and the length of how far away the clicked point is from the bottom left point.

  • Acceleration
  • Velocity
  • Mass
  • Speed
